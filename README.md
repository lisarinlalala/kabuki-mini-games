# Kabuki Mini Games

歌舞伎を短いブラウザゲームで体験するシリーズです。

## 収録ゲーム

- `classic/`：引窓 Classic（スワイプ操作）
- `easy/`：引窓 Easy（タップ操作・見得で停止）
- `index.html`：ゲーム選択メニュー

## GitHubへのアップロード

現在のリポジトリのトップ階層へ、以下をアップロードしてください。

- `index.html`
- `classic` フォルダ
- `easy` フォルダ
- `README.md`

同名の `index.html` は、今回のメニュー画面に置き換えます。
元のゲームは `classic/index.html` に保存されているため、消えません。
